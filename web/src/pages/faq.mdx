# FAQ

（還沒有人問的時候叫做 _Fully Anticipated Questions_，我覺得大概會有人問的問題。）

## 用詞

- 台語 / 臺灣閩南語 / 福建話 …：我目前傾向「台語」，但也可以理解稱之為「臺灣閩南語」（閩南語這個語言的臺灣分支）的緣由。閩南語整體都是[需要復興的](https://www.speakhokkien.org/hokkien)。
- 「國語」 / 華語：稱作華語。「[臺灣各固有族群使用之自然語言及臺灣手語](https://law.moj.gov.tw/LawClass/LawAll.aspx?pcode=H0170143)」都是國家語言。在這方面 Kemdict 還需要想辦法包含各族語和臺灣手語。

## 其他

- 為何目前沒有包含各原住民族語：我只知道原民會的[原住民族語言線上辭典](https://e-dictionary.ilrdf.org.tw)，而我還沒嘗試向其請求授權。
- 為何沒有包含臺灣手語：一部分是我還沒找資源，另一部分是我在重新讀國家語言發展法之前忘了它的存在。
- 台語該用白話字還是教育部臺羅：在 Kemdict 我選擇了一律使用教育部臺羅，但搜尋時也支援傳統白話字。後者只是白話字這個書寫系統的像是[德文正寫法](https://en.wikipedia.org/wiki/German_orthography_reform_of_1996)改革一樣的拼字改革 — 雖然[不是很有足夠共識](https://www.thenewslens.com/article/28741)，但實際使用上應該是新的拼法比較多人會比較熟悉。
